# Web Design Portfolio

## Projects

### 1. MoePet Home - Pet Adoption Platform
A comprehensive pet adoption and service platform featuring pet adoption, community forums, online store, AI smart customer service, and health records management.

- **Tech Stack**: HTML5, CSS3, JavaScript, Node.js, Express, SQLite
- **Features**: Pet adoption management, AI chatbot, community forums, e-commerce, health records, lost & found, grooming appointment, face recognition login, admin dashboard

### 2. Digital Accessories Mall
A full-featured e-commerce platform for digital accessories with product browsing, shopping cart, order management, user center, and admin backend.

- **Tech Stack**: Spring Boot, Thymeleaf, MyBatis, MySQL, Java
- **Features**: Product categories, carousel banners, flash sales, shopping cart, order system, user auth, product search, reviews, admin management
